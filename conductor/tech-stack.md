# Technology Stack

## Core Technologies
- **Framework**: Angular 21 (Leveraging Signals, new input/output APIs, and CSS Grid).
- **Language**: TypeScript (Strict typing for all data structures and component APIs).
- **Component Architecture**: Strict separation of concerns. Templates and styles MUST be kept in separate external files (never inline in the component class).
- **Styling**: SCSS (For any component-specific styling or default structures).

## Workspace & Build Tools
- **CLI**: Angular CLI (Standard, official tooling for managing the library and demo app).
- **Package Manager**: npm.

## Testing Frameworks
- **Unit Testing**: Vitest (A modern, extremely fast test runner).
- **End-to-End Testing**: Playwright (A modern, reliable multi-browser testing framework for the demo app).
